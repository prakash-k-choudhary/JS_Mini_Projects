# JS Mini Practice Projects

A collection of JavaScript mini projects built for practicing core frontend development concepts including DOM manipulation, event handling, dynamic UI updates, and responsive design using HTML, CSS, and Vanilla JavaScript.

---

## 🚀 Projects Included

### 🎨 Theme Switcher
A simple theme switching application that dynamically changes the background colors and UI theme using JavaScript event handling.

### 🧮 BMI Calculator
A BMI calculator that takes user input for height and weight, calculates Body Mass Index, and displays the health category dynamically.

### ⏰ Digital Clock
A real-time digital clock built using JavaScript Date functions that updates every second.

---

## 🛠️ Tech Stack

- JavaScript (ES6)

## 📚 Concepts Practiced
